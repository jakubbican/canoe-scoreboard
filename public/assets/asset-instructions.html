Scoreboard Asset Setup Instructions

You need to provide the following files for the scoreboard to display correctly:

Required Files

  • logo.png - The main logo displayed in the top bar
  • footer.png - The footer image with sponsor logos
  • bib.png - The background image for competitor bibs
  • partners.png - Partners/sponsors logo image (optional)
  • flags/*.bmp - Country flag BMP files (named as country codes, e.g., CZE.bmp)
  • flags/empty.bmp - A transparent/empty flag for when no country is specified

How to Add These Files

  1. Create a folder named assets-source in the project root
  2. Create a subfolder named flags inside assets-source
  3. Copy your asset files to these folders
  4. Run node scripts/prepare-assets.js to process the assets

After adding your assets, rebuild the application with npm run build.
